# ionic-pouchdb-couchdb-sync-seed

This is a working spike solution for using ionic, pouchdb, couchdb to facilitate 4 way sync.

P.S. : Doesn't work/breaks with pouchdb.js over 3.2.0 (tested with 3.6.0 and 5.1.0).
